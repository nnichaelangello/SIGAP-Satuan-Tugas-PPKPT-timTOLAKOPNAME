================================================================================
         SKENARIO TEST CHATBOT - KASUS PELECEHAN BERBELIT-BELIT
================================================================================

Tujuan: Menguji kemampuan chatbot mendeteksi intent pelecehan meskipun
        user tidak langsung cerita dan berbelit-belit.

Ekspektasi:
- Chatbot harus tetap sabar dan empatik
- Skor intent harus naik seiring cerita terungkap
- Chatbot harus memicu consent ketika skor >= 7

================================================================================
                   CATATAN UNTUK TESTER
================================================================================

1. CATAT SKOR di setiap response chatbot
   - Lihat parameter "score" di JSON response
   - Lihat parameter "tier" untuk level data persistence

2. PERHATIKAN ESCALATION
   - Skor harus naik seiring cerita terungkap
   - Consent harus muncul di skor >= 7

3. CEK SENSITIVITAS
   - Chatbot tidak boleh langsung tanya "apakah kamu dilecehkan?"
   - Harus tunggu user yang cerita sendiri
   - Tetap empatik meski user berbelit-belit

4. CEK EMERGENCY
   - Kata kunci darurat harus langsung trigger hotline
   - Emergency log harus tercatat di database

5. VERIFIKASI LOCAL FALLBACK
   - Matikan internet, chatbot harus tetap bisa respons
   - Respons mungkin lebih sederhana tapi tetap empatik
